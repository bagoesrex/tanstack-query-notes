# Queries

A **query** is a declarative dependency on an asynchronous data source tied to a **unique key**. It automatically manages data fetching, caching, background updates, and error handling so you don’t need to manually use `useState` and `useEffect`.

To subscribe to a query in your components or custom hooks, call the `useQuery` hook with at least:

- A **unique key for the query**
- A function that returns a promise that:
  - Resolves the data, or
  - Throws an error

```tsx
import { useQuery } from "@tanstack/react-query";

function App() {
  const info = useQuery({
    queryKey: ["todos"], // unique key
    queryFn: fetchTodoList, // function that returns a promise
  });
}
```

## Query States

A query can only be in one state at a time:

| State                               | Meaning                               |
| ----------------------------------- | ------------------------------------- |
| `isPending or status === 'pending'` | The query has no data yet             |
| `isError or status === 'error'`     | The query failed to fetch data        |
| `isSuccess or status === 'success'` | The query was successful and has data |

example:

```tsx
if (isPending) return <span>Loading...</span>;
if (isError) return <span>Error: {error.message}</span>;
return <DataView data={data} />;
```

## Fetch Status

Besides `status`, each query also has a `fetchStatus` that tracks fetching activity:

| Fetch Status                 | Description                                                       |
| ---------------------------- | ----------------------------------------------------------------- |
| `fetchStatus === 'fetching'` | The query is currently fetching (including background refetching) |
| `fetchStatus === 'paused'`   | The query wanted to fetch but is paused (e.g., offline)           |
| `fetchStatus === 'idle'`     | The query is not doing anything                                   |

## Rule of thumb

- `status` → describes the **data** (do we have it or not?)
- `fetchStatus` → describes the **queryFn** (is it running or not?)
