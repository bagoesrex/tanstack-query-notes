# Background Fetching Indicators

Besides showing initial loading (`status === 'pending'`), you can use `isFetching` to indicate **background refetching** - when data updates silently in the background.

```tsx
const { status, data, error, isFetching } = useQuery({
  queryKey: ["todos"],
  queryFn: fetchTodos,
});

return status === "pending" ? (
  <span>Loading...</span>
) : status === "error" ? (
  <span>Error: {error.message}</span>
) : (
  <>
    {isFetching && <div>Refreshing...</div>}
    {data.map((todo) => (
      <Todo todo={todo} />
    ))}
  </>
);
```

## Global Indicator

To show a loading state for **any background query**, use `useIsFetching()`:

```tsx
import { useIsFetching } from "@tanstack/react-query";

function GlobalLoadingIndicator() {
  const isFetching = useIsFetching();

  return isFetching ? (
    <div>Queries are fetching in the background...</div>
  ) : null;
}
```
